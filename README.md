# 📊 AI Student Impact Analysis

An end-to-end Data Analytics project that analyzes how Generative AI influences students' academic performance, learning behavior, and well-being. The project combines Python, PostgreSQL, SQL, and Power BI to transform raw data into actionable insights.

---

## 🎯 Project Objective

The primary objective of this project is to:

- Analyze students' academic performance before and after AI usage.
- Study AI usage patterns across different academic disciplines.
- Compare learning behavior between paid and non-paid AI users.
- Examine the relationship between AI usage, anxiety level, and burnout risk.
- Present insights through an interactive Power BI dashboard.

---

## 📊 SQL Analysis

Some of the business questions answered include:

- Number of students in each major category.
- Average Pre and Post Semester CGPA by major.
- Impact of AI usage on academic performance.
- Average weekly AI usage by students.
- Comparison between paid and non-paid AI users.
- Students facing high anxiety levels across majors.
- Major category contributing the highest number of top-performing students.

## 💡 Key Insights

- Average Post-Semester CGPA exceeded the Pre-Semester CGPA, indicating an overall improvement in academic performance.
- Students with paid AI subscriptions spent more time using Generative AI tools.
- AI usage patterns varied across different years of study and academic disciplines.
- Prompt engineering skills differed significantly across major categories.
- Student well-being indicators such as anxiety level and burnout risk showed noticeable variation across groups.
